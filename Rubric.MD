Each criteria carries 5 pts each
### Criteria 1: User Flow & UI/UX (ease of use)
Description:
How intuitive and smooth the experience is for a first-time user: clear entry point, sensible steps, understandable outputs, good interaction design.
Score (1–5 guide)
1: confusing/hard to use
2: functional but clunky
3: clear flow but less smooth
4-5: intuitive + polished, fast to learn

### Engineering Quality (reliability + integration)
Description:
How well it’s engineered: clean integration of components/tools/models/data, sensible architecture, and resilience to common failure modes, working end to end
Score (1-5 guide):
1: brittle; breaks easily 
2: works but fragile; many manual steps
 3: solid end-to-end build; components wired well
 4-5: robust + well-architected; handles edge cases;

### Grounding & Verifiability (MVP level)
Description:
Key outputs traceable to inputs/tool results. Lightweight “show work” (citations/links/highlights/structured extraction/log). Avoids confident claims without evidence; flags uncertainty.

Score (1-5 guide):
1: unverifiable 
2: some provenance  
3: most key claims traceable 
4-5: strong lightweight verification + uncertainty handling


### Creativity, style & Innovation. 

Description:
Distinct creative point of view and a fresh twist: cohesive tone/voice, pleasing visuals, and outputs that feel intentional and “on theme”. Innovation can be a novel format, interaction, or idea that improves the experience—not just decoration.

Score (1-5 guide):
1: generic/flat
2: small twist but some style. 
3: clear differentiator 
4-5: memorable innovation, polished + works in demo

### Use Case & Real-World Fit
Description:
Clear user + use case, tight scope, and demo scenario matches the problem. Output is actionable (e.g., study guide+quiz, bracket what-ifs, highlights, recipe plan, game assets). Basic constraints/assumptions are acknowledged

Score (1-5 guide):
1: unclear story/value
2: decent pitch, partial flow 
3: clear use case + complete thin slice 
4-5: compelling, scoped, immediately useful
